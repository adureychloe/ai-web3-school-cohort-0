# AI × Web3 School Learning Repo

Personal learning journal and proof-of-work for AI × Web3 School.

Links
- Handbook: https://aiweb3.school/zh/handbook/
- WCB Program: https://web3career.build/zh/programs/AI-Web3-School
- WCB Learning: https://web3career.build/zh/programs/AI-Web3-School#tab=learning

Privacy reminder
- This repository is public by default.
- Do not commit API keys, private keys, seed phrases, passwords, private contact details, private meeting links, or other sensitive data.

Repository structure
- profile.md: learner profile and goals
- learning-plan.md: current learning plan and milestones
- daily/: daily study notes and check-ins
- tasks/: task breakdowns and mini-assignments
- experiments/: prototypes, prompts, demos, and test artifacts
- handbook-feedback/: handbook issues, suggestions, and source-linked feedback
- hackathon/: project ideas, drafts, and build notes
- submissions/: links and records for official submissions
- templates/: reusable daily/task note templates

Working style
- Keep notes small, concrete, and linkable.
- Prefer proof-of-work over abstract summaries.
- Record blockers, fixes, references, and next steps.
