# Handbook Feedback Workflow

This directory stores public, linkable feedback for AI × Web3 School Handbook.

Each feedback item should include:
- page link
- issue type
- problem description
- suggested fix
- source from learning process

Suggested issue types
- typo
- broken/outdated link
- unclear concept
- missing example
- missing prerequisite
- structure/navigation suggestion

Workflow
1. Capture confusion during study in daily notes
2. Promote real issues into a dedicated feedback file here
3. Keep one issue per file when possible
4. Add page URL and concrete suggestion
5. Reference the daily note that triggered the feedback
6. If adopted or resolved, mark status later
