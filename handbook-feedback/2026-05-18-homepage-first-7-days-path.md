# Feedback: Add a beginner-friendly first 7 days path

Status: draft
Page: https://aiweb3.school/zh/handbook/
Source: daily/2026-05-18.md
Type: structure/navigation suggestion

## Problem
The homepage explains the overall value of AI × Web3 School clearly, but a new learner may still not know what to read first over the first week.

## Why this matters
A short guided path would reduce startup friction and help learners convert interest into action.

## Suggested improvement
Add a small “First 7 Days” section on the homepage with:
- Day 1: Handbook home
- Day 2: LLM
- Day 3: Prompt
- Day 4: Context
- Day 5: Wallet
- Day 6: Network
- Day 7: Agent or Smart Contract

Include estimated time per day and expected output format, such as note / diagram / question list.
