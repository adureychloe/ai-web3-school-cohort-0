## What I learned

- Wallet is not just a Web3 login button. It is the boundary where users manage accounts, connect addresses, sign messages, send transactions, switch networks, and confirm risks.
- Connecting a wallet does not mean the application controls the account. The user still owns the account boundary, and every signature or transaction needs a clear explanation.
- A seed phrase and private key are high-risk secrets. No website, form, customer support, AI agent, or script should ask for them.
- Network is the execution environment for blockchain state changes. Mainnet, testnet, and L2 have different costs, speeds, security assumptions, explorers, and contract addresses.
- A transaction is not the same as clicking a button. The button only starts the request; the transaction still needs wallet confirmation, signature, broadcast, inclusion in a block, execution, and confirmation.
- Gas is the cost of using network execution and storage resources. A product should explain the estimated fee, fee asset, possible failure, and whether failed transactions may still consume gas.
- Block explorers are verification tools. They help users and developers check transaction status, address activity, contract events, and onchain facts.
- Smart contracts are public programs that hold state and rules. They can be composed by other apps, but this openness also creates security and permission risks.

## AI × Web3 connection

The most important connection today is that an AI agent can help explain, prepare, and verify Web3 actions, but it should not silently execute them. In a Web3 workflow, the agent needs explicit boundaries:

1. It can read public docs, ABI, explorer pages, and local notes.
2. It can summarize risks, estimate the likely action path, and draft instructions.
3. It must not ask for seed phrases, private keys, or raw secrets.
4. It must not auto-sign messages, approve tokens, send transactions, or deploy contracts without explicit user confirmation.
5. It should show the network, address, contract, action, estimated gas, and possible side effects before any wallet step.
