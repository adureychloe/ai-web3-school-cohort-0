### 3️⃣ Web UI Backend 接入

修复 `BrokenPipeError` 根因：`asyncio.to_thread` 中 `print()` 在 web 环境 stdout 关闭。

修复方案：`to_thread()` 函数改为捕获 stdout 到 `io.StringIO()`，防止 BrokenPipeError 传播。

所有 API 端点测试通过（全部 200 OK）：

- `GET /api/proofs` — 1 笔交付存证
- `POST /api/pay` — Pact → Transfer → Proof

新增 API 端点：

## 关键技术决策

1. **Procurement Agent 匹配策略**：英文关键词重叠 + 中文标签桥接。`_CN_TAGS` per service ID，中文字符单个匹配，英文词 3+ 字符匹配。完美处理中英文混合查询。
2. **print() -> _log()**：为防止 asyncio 线程 BrokenPipeError，改用 `sys.stderr` + try/except。
3. **json_safe 自动 sanitize**：递归过滤 `api_key` 等敏感字段，所有 API 端点自动受益。
4. **CAW 凭证** 始终 auto-discover from `~/.cobo-agentic-wallet/`，不在代码中硬编码。
