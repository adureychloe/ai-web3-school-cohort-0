# AI × Web3 School Learning Plan

## Profile summary
- AI: 有基础
- Web3: 新手
- Coding: 会基础脚本
- Goal: Hackathon-oriented
- Time budget: 1 hour/day
- Language: Chinese + English

## 2-week initialization plan

### Week 1: Build foundations and vocabulary
1. Read Handbook overview and understand the map
2. Focus AI basics that matter for agents:
   - LLM
   - Prompt
   - Context
   - Agent
3. Focus Web3 basics needed for practical understanding:
   - Wallet
   - Network
   - Smart Contract
4. Write one short note per topic
5. Track every confusion point in handbook-feedback/

Expected outputs
- 4 to 6 daily notes
- 1 concept map
- 3+ handbook feedback items or open questions

### Week 2: Bridge AI and Web3 toward project ideas
1. Read Bridge sections most relevant to builders:
   - Chain-aware Context
   - Web3 Tool Use
   - Agent Workflow
   - Agent Wallet
   - Machine Payment
2. Turn notes into project ideas
3. Compare 2 to 3 hackathon directions
4. Pick one starter project and define MVP

Expected outputs
- 1 project shortlist
- 1 MVP draft
- 1 implementation sketch in hackathon/

## Daily study path (1 hour)
- 10 min: read today’s target page
- 15 min: summarize key concepts in your own words
- 15 min: connect concept to a real use case or product idea
- 10 min: log blockers / unclear wording / outdated links
- 10 min: write check-in draft and next action

## Three-track mode for each day
- Minimum path: finish one core concept + 3 bullet summary
- Recommended path: add one comparison, example, or diagram
- Challenge path: add a small experiment, prototype, or prompt workflow

## Hackathon goal framing
By the end of the first phase, choose one direction:
- AI agent for onchain operations
- wallet / permission UX for agents
- AI + Web3 developer tooling
- verifiable / secure AI workflow onchain
- autonomous commerce / payment flow

## Success criteria
- I can explain the core AI × Web3 primitives in simple words
- I have a public proof-of-work trail
- I can point to one concrete hackathon direction
- I maintain a feedback loop back to the Handbook
